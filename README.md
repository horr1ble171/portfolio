# horr1ble - [Портфолио](https://www.horr1ble.space/) Full-Stack Разработчика

Мое личное портфолио, демонстрирующее широкий спектр проектов, от full-stack веб-разработки до продвинутых Telegram и Discord ботов. Я сосредоточен на создании цифровых продуктов от концепции до развертывания, придерживаясь продуктового подхода.

## ✨ Ключевые особенности

*   **Современный и адаптивный дизайн:** Создан с акцентом на пользовательский опыт и адаптивность на различных устройствах.
*   **Разнообразная витрина проектов:** Представлен широкий спектр проектов, включая веб-приложения, Telegram-боты и инструменты мониторинга серверов.
*   **Комплексный набор навыков:** Демонстрирует владение как фронтенд, так и бэкенд разработкой, DevOps и управлением продуктами.
*   **Продуктовый подход:** Каждый проект разрабатывается с акцентом на решение реальных проблем пользователей и достижение органического роста.
*   **Интерактивный UI/UX:** Используются современные библиотеки для плавной анимации и интерактивных элементов.
*   **Многоязычная поддержка:** Сайт разработан для поддержки нескольких языков.

## 🚀 Избранные проекты

Вот некоторые из моих проектов:

*   **Музыкальный бот:** Telegram-бот, который позволяет пользователям легко редактировать метаданные аудиофайлов (обложка, название, исполнитель, альбом) прямо в Telegram.
*   **Beam Builds:** (В разработке) Платформа для публикации и поиска пользовательских сборок модов для игры BeamNG.drive.
*   **Syncly:** (В разработке) Платформа, предназначенная для бесшовной миграции музыкальных плейлистов между различными потоковыми сервисами (YouTube Music, Deezer, SoundCloud) с использованием сессионных данных для обхода ограничений официальных API.
*   **AntiX Server Monitor:** Веб-интерфейс на Python для мониторинга в реальном времени CPU, RAM, диска, сети, температуры и топовых процессов на домашнем сервере Linux AntiX.
*   **Сайт-портфолио:** Сам сайт, который вы сейчас просматриваете, созданный для демонстрации моих навыков и проектов с акцентом на современный дизайн и интерактивность.
